# auto-ls.fish

A fish shell plugin to run `ls` when no command is given

## Install

[reel] installation:

```shell
reel in mattmc3/auto-ls.fish
```

[fisher] installation:

```shell
fisher add mattmc3/auto-ls.fish
```

## Inspiration

- Zsh has a similar plugin [here][auto-ls].
